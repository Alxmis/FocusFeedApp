# FocusFeed

FocusFeed — это iOS-приложение, помогающее использовать социальные платформы более осознанно и без лишних отвлечений.

## Описание

Современные цифровые платформы часто содержат элементы, которые затягивают внимание и мешают сосредоточиться.  
FocusFeed настраивает отображение публично доступного веб-контента, чтобы уменьшить отвлекающие элементы и помочь пользователю сосредоточиться на действительно важном.

Приложение не является альтернативным клиентом и не предоставляет собственный контент.  
Вся информация отображается напрямую с публичных веб-страниц.

## Возможности

- Более спокойный и сфокусированный просмотр контента  
- Отсутствие отвлекающих коротких видеолент  
- Локальная статистика использования  
- Простые настройки под пользователя  

## Конфиденциальность

FocusFeed уважает конфиденциальность пользователей.

- Приложение **не собирает** персональные данные  
- Приложение **не использует** аналитику и сторонние трекеры  
- Все данные и настройки хранятся **локально на устройстве**  
- Взаимодействие с веб-сайтами происходит напрямую между пользователем и соответствующим сайтом

## Техническая информация

- Платформа: iOS  
- Приложение использует `WKWebView` для отображения публично доступного веб-контента  
- Не требует регистрации или входа в аккаунт
